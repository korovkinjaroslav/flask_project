# Техническое задание  
## Проект: Веб-приложение «MiniReddit» на Flask

---

## 1. Общая информация  

**Название проекта:** MiniReddit  

**Цель:**  
Разработка веб-приложения для публикации постов и общения пользователей в формате обсуждений.

**Задачи проекта:**
- Реализовать базовую систему постов и комментариев  
- Добавить регистрацию и авторизацию пользователей  
- Создать простой и удобный интерфейс  

---

## 2. Пользователи системы  

Система предполагает два типа пользователей:

### 🔹 Гость (неавторизованный пользователь)
- Просмотр постов  
- Просмотр комментариев  

### 🔹 Авторизованный пользователь
- Создание постов  
- Написание комментариев  
- Лайк / дизлайк постов  
- Удаление своих постов и комментариев  

---

## 3. Функциональные требования  

### 3.1. Регистрация и авторизация  
- Регистрация по логину и паролю  
- Авторизация (вход в систему)  
- Выход из аккаунта  

---

### 3.2. Работа с постами  

Пользователь может:
- Создавать пост:
  - заголовок  
  - текст  
- Просматривать список постов (лента)  
- Открывать отдельный пост  
- Удалять свои посты  

**Дополнительно:**
- Сортировка (по дате или популярности)  

---

### 3.3. Комментарии  
- Добавление комментариев к посту  
- Отображение списка комментариев  
- Удаление своих комментариев  

---

### 3.4. Лайки / рейтинг  

Пользователь может:
- Поставить лайк  
- Убрать лайк  

У поста отображается общее количество лайков  

---

## 4. Структура базы данных  

### Таблица `users`
- `id`  
- `username`  
- `password`  

### Таблица `posts`
- `id`  
- `title`  
- `content`  
- `user_id`  
- `created_at`  

### Таблица `comments`
- `id`  
- `content`  
- `user_id`  
- `post_id`  
- `created_at`  

### Таблица `likes`
- `id`  
- `user_id`  
- `post_id`  

---

## 5. Технологии  

- **Backend:** Python + Flask  
- **База данных:** SQLite *(может быть другое)*  
- **ORM:** SQLAlchemy  
- **Frontend:**  
  - HTML  
  - Bootstrap  

---

## 6. Интерфейс  

### Основные страницы:
1. Главная страница (лента постов)  
2. Страница поста (пост + комментарии)  
3. Страница входа  
4. Страница регистрации  
5. Страница создания поста  

---

## 7. Возможные улучшения  

*(при наличии дополнительного времени)*

- Подписки на пользователей  
- Поиск постов  
- Теги постов  

---

## 8. Ожидаемый результат  

Веб-приложение, в котором:
- Пользователи могут регистрироваться  
- Создавать посты  
- Обсуждать их в комментариях  
- Взаимодействовать через систему лайков  

---

## Итог  

Проект «MiniReddit» представляет собой упрощённую версию социальной платформы для обсуждений, реализованную с использованием Flask и современных веб-технологий.
